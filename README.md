# publisher-subscribe

## :rocket: Começando

```bash
  # Clonar o projeto:
  $ https://github.com/Pedroor/publisher-subscribe

  # Antes de continuar é preciso ter o nodejs e o npm instalado na sua máquina:
  $ Linux - https://linuxize.com/post/how-to-install-node-js-on-ubuntu-18.04/
  $ Windows and MacOs - https://radixweb.com/blog/installing-npm-and-nodejs-on-windows-and-mac

  # Entrar no diretório:
  $ cd publish-subscribe

  # Instalar as dependências:
  $ npm install

  # Abrir no mínimo 2 terminais 1 para o Publisher e outro para a quantidade de Subscribers que desejam receber as informações:

  # No primeiro terminal execute
  $ cd Publisher
  $ node index.js

   # No segundo terminal execute
  $ cd Subscribe
  $ node index.js

```
